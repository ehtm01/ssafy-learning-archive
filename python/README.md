### [3. Algorithms](./algorithms)
다양한 자료구조와 알고리즘 문제 해결 기법을 다룹니다.

#### [List & String](./algorithms/list-string)
배열과 문자열 알고리즘
- 1차원/2차원 배열
- 문자열 탐색 및 패턴 매칭
- 완전 탐색, 순열, 조합

#### [Stack & Queue](./algorithms/stack-queue)
스택과 큐 자료구조
- 스택 기초 및 응용
- 큐, 덱(Deque)
- 우선순위 큐

#### [Tree](./algorithms/tree)
트리 자료구조
- 이진 트리
- 트리 순회 (전위, 중위, 후위)
- 이진 탐색 트리

#### [Graph](./algorithms/graph)
그래프 알고리즘
- 그래프 표현 (인접 행렬, 인접 리스트)
- DFS, BFS
- 최단 경로 (다익스트라, 벨만-포드)
- 최소 신장 트리 (크루스칼, 프림)
