# Python 학습 요약

## 학습 개요
Python 프로그래밍의 기초부터 자료구조, 알고리즘까지 체계적으로 학습

## 1. Python 기초

### 핵심 개념
- **프로그래밍 기본**: 변수, 데이터 타입, 연산자
- **제어 구조**: 조건문(if/elif/else), 반복문(for/while)
- **함수**: 함수 정의, 매개변수, 반환값, 스코프
- **시퀀스 타입**: 문자열, 리스트, 튜플
- **문자열 처리**: 인덱싱, 슬라이싱, f-string

### 학습 키워드
```
변수 | 데이터 타입 | 메모리 | 연산자
조건문 | 반복문 | 함수 | 스코프
문자열 | 리스트 | 튜플 | 딕셔너리
```

## 2. 자료구조 및 OOP

### 핵심 개념
- **자료구조**: List, Dictionary, Tuple, Set
- **메서드**: 문자열 메서드, 리스트 메서드
- **복사**: 얕은 복사(Shallow Copy) vs 깊은 복사(Deep Copy)
- **객체지향 프로그래밍(OOP)**:
  - 클래스와 인스턴스
  - 캡슐화, 상속, 다형성
  - 매직 메서드(`__init__`, `__str__`, etc.)
- **예외 처리**: try/except, 예외 종류

### 학습 키워드
```
자료구조 | 메서드 | 얕은복사 | 깊은복사
클래스 | 인스턴스 | 상속 | 다형성
캡슐화 | 예외처리 | try-except
```

## 3. 알고리즘

### List & String
**핵심 개념**
- 1차원/2차원 배열
- 완전 탐색(Brute Force)
- 문자열 패턴 매칭
- 순열(Permutation)과 조합(Combination)

**학습 키워드**: `배열 | 문자열 탐색 | 완전탐색 | 순열 | 조합`

### Stack & Queue
**핵심 개념**
- 스택(Stack): LIFO 구조
- 큐(Queue): FIFO 구조
- 덱(Deque): 양방향 큐
- 우선순위 큐(Priority Queue)

**학습 키워드**: `스택 | 큐 | 덱 | 우선순위큐 | LIFO | FIFO`

### Tree
**핵심 개념**
- 이진 트리(Binary Tree)
- 트리 순회: 전위(Preorder), 중위(Inorder), 후위(Postorder)
- 이진 탐색 트리(BST)

**학습 키워드**: `트리 | 이진트리 | 트리순회 | BST`

### Graph
**핵심 개념**
- 그래프 표현: 인접 행렬, 인접 리스트
- 탐색 알고리즘: DFS, BFS
- 최단 경로: 다익스트라(Dijkstra), 벨만-포드(Bellman-Ford)
- 최소 신장 트리: 크루스칼(Kruskal), 프림(Prim)

**학습 키워드**: `그래프 | DFS | BFS | 다익스트라 | 최소신장트리`
