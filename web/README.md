# Web Development

SSAFY 1학기에서 학습한 웹 개발 관련 내용을 정리한 폴더입니다.

## 목차

## Frontend

### [HTML/CSS](./frontend/html-css)
웹의 기본 구조와 스타일링을 학습합니다.

**주요 내용:**
- HTML 기본 태그 및 시맨틱 태그
- CSS 선택자, Box Model
- Flexbox & Grid Layout
- Bootstrap 컴포넌트
- 반응형 웹 디자인

**학습 파일:**
- `0825_Web.md` - HTML/CSS 기초
- `0826_CSS_Layout.md` - CSS 레이아웃
- `0827_Bootstrap.md` - Bootstrap 기초
- `0828_Responsive_Web.md` - 반응형 웹

---

### [JavaScript](./frontend/javascript)
웹 페이지에 동적 기능을 추가하는 JavaScript를 학습합니다.

**주요 내용:**
- DOM 조작
- 변수, 함수, 객체
- 이벤트 처리
- Ajax & Fetch API
- JSON 데이터 처리

**학습 파일:**
- `1117_JS_DOM.md` - DOM 조작
- `1118_JS_BasicSyntax_01.md` - JS 기본 문법 1
- `1119_JS_BasicSyntax_02.md` - JS 기본 문법 2
- `1120_JS_ControllingEvent.md` - 이벤트 제어
- `1124_JS_Ajax.md` - Ajax 통신
- `1125_JS_Ajax_with_Django.md` - Django와 Ajax 연동

---

### [Vue.js](./frontend/vue)
프론트엔드 프레임워크 Vue.js를 학습합니다.

**주요 내용:**
- Vue 인스턴스
- 디렉티브 (v-if, v-for, v-model 등)
- 컴포넌트 기초
- Single File Component (SFC)
- Props & Emit
- Component 간 데이터 흐름

**학습 파일:**
- `1126_Vue_Introduction_of_vue.md` - Vue 소개
- `1127_Vue_BasicSyntax_01.md` - Vue 기본 문법 1
- `1201_Vue_BasicSyntax_02.md` - Vue 기본 문법 2
- `1202_Vue_SingleFileComponents.md` - SFC
- `1203_Vue_ComponentStackFlow.md` - 컴포넌트 데이터 흐름

## 학습 순서 추천

### Frontend 로드맵
1. **HTML/CSS** - 웹 구조와 스타일 기초
2. **JavaScript** - 동적 웹 페이지 구현
3. **Vue.js** - 프론트엔드 프레임워크

### Backend 로드맵
1. **Django** - 웹 프레임워크 기초
2. **Django REST Framework** - API 서버 개발

### Full Stack 통합
- Django (Backend) + Vue.js (Frontend)
- RESTful API 통신
- 인증 및 권한 관리
