논리와 증명 - 문제 12: n^2이 3의 배수이면 n은 3의 배수임을 증명하라

Proof)
대우: n이 3의 배수가 아니면 n^2는 3의 배수가 아니다.
1. n = 3k + 1 일 때, 
    n^2 = 9k^2 + 6k + 1
        = 3(3k^2 + 2k) + 1 이므로 3의 배수가 아니다.

2. n = 3k + 2 일 때,
    n^2 = 9k^2 + 12k + 4
        = 3(3k^2 + 4k + 1) + 1 이므로 3의 배수가 아니다.

대우가 참이므로, n^2이 3의 배수이면 n은 3의 배수이다.

기초 수식 - 문제 4: T(n) = T(n/2) + 1, T(1) = 1

Proof)
T(n) = T(n / 2) + 1
     = T(n / 2^2) + 1 + 1
     = T(n / 2^k) + k
     = T(1) + logn, n == k

O(T(n)) = O(logn)
